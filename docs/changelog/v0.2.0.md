# Unreleased

## Added

- **タスク進行順序（order）機能**: タスクに階層的な進行順序を設定し、作業の優先順位を管理できる機能を追加
  - `tm new --order <value>`: 新規タスク作成時にorderを設定
  - `tm update --order <value>`: 既存タスクのorderを更新（`null`で解除）
  - `tm list --sort <key>`: ソート順を指定（`order`（デフォルト）、`id`、`created`）
  - フォーマット: `1`, `2`, `1-1`, `1-2`, `1-2-1` など（ハイフン区切りで親子関係を表現）
  - 小数入力のサポート（例: `1.5`）と自動正規化
  - `todo`、`wip` ステータスのみorderを保持、それ以外では自動的に `null`

## Changed

- **デフォルトソート順を変更**: `tm list` のデフォルトソートが作成日時順からorder昇順に変更
  - orderが未設定のタスクは後ろに表示
  - 従来の動作に戻すには `--sort created` または `--sort id` を使用
